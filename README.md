# graph-algos
A collection of problems on graphs for technical interviews. Such problems are based on the excellent tutorial [https://www.youtube.com/watch?v=tWVWeAqZ0WU](https://www.youtube.com/watch?v=tWVWeAqZ0WU).

## Exercises
1. Depth-first search
2. Breadth-first search
3. Has path
4. ...
5. Connected components
6. Largest component
7. Shortest path
8. Number of islands
9. Size of min insland
